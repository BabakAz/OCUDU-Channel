# Release Roadmap

## v0.1

Release when the project has:

- native Ubuntu 22.04+ build instructions;
- localhost 1 gNB x 1 UE pass-through demo;
- deterministic channel-model examples;
- benchmark CSV output;
- CUDA MVP latency and throughput tables with CPU reference baselines;
- known-limits documentation for ZMQ/srsUE scale.

## v0.2

Release after:

- validated 1 gNB x 3 UE local runs;
- distributed network feasibility results on wired Ethernet;
- expanded CUDA model coverage with documented CPU reference comparison.

## v1.0

Release only after repeatable validation across at least two machines or two lab topologies.
